Privacy Policy

CpT Móvil  /  Arduino Data

Last updated: May 2026

What data this app collects

None. CpT Móvil does not collect, store, or transmit any personal information. No accounts, no sign-in, no analytics.

Sensor data

Sensor readings received from a connected Arduino are displayed on screen and held in your device's memory only for the duration of your session. They are never sent to any server. If you export a CSV file, that file is shared only by your own deliberate action using your device's standard sharing tools (email, Drive, etc.) and goes only where you send it.

Third-party services

The app does not include advertising SDKs, crash-reporting services, analytics libraries, or any other third-party code that collects data.

Children

This app is designed for use in educational settings, including by students under 13. Because no data is collected, it is fully compliant with COPPA and similar regulations.

Changes to this policy

If this policy ever changes, the updated version will be posted at this URL with a new "Last updated" date.
